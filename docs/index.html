Coin Price Backend Project

Ứng dụng backend xử lý dữ liệu giá tiền mã hóa

Thông tin dự án

  • Ngôn ngữ: Java
  • Cơ sở dữ liệu: MongoDB
  • Công nghệ: Spring Boot

Thư viện sử dụng và tác dụng

  • Spring Boot Starter Web: Xây dựng các API RESTful cho ứng dụng web.
  • Spring Boot Starter Data JPA: Hỗ trợ thao tác với cơ sở dữ liệu thông qua JPA (Java Persistence API).
  • Spring Boot Starter Security: Tích hợp bảo mật cho các API, bao gồm xác thực và phân quyền.
  • Caffeine Cache Management: Quản lý bộ nhớ đệm (cache) hiệu quả cho các yêu cầu dữ liệu.
  • Spring Boot Starter WebSocket: Hỗ trợ giao tiếp thời gian thực với WebSocket, giúp cập nhật dữ liệu nhanh chóng.
  • Spring Boot Starter Data MongoDB: Tương tác với cơ sở dữ liệu MongoDB, cho phép lưu trữ và truy xuất dữ liệu.
  • Lombok: Giảm thiểu mã boilerplate bằng cách tự động sinh ra getter, setter và các phương thức khác.
  • Nimbus JOSE + JWT: Hỗ trợ xử lý mã hóa và bảo mật với JSON Web Tokens (JWT) cho các yêu cầu xác thực.
  • Swagger UI: Tạo tài liệu API trực quan và dễ dàng kiểm thử các API trực tiếp từ giao diện web.
  • Java Dotenv: Quản lý các biến môi trường trong ứng dụng Java, giúp dễ dàng cấu hình các thông tin nhạy cảm.
